# fract-ol (work only on macOS)
 Displays the famous fractals of Julia and Mandelbrot
## How to use?
1) `make`
2) `./fractol number_of_parameter`  
 
   1 - Julia Set  
   2 - Mandelbrot Set  

## Control
Movement - `W A S D`  
Zooming in and out - `Mouse scrool`  
Changing color - `shift`  
Exit program - `esc`
### for Julia
Buttons `1 2 3 4` - to change shape
